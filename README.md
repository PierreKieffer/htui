## Install 

**Note**: Prebuilt binaries for 64-bit operating systems, doesn't require Go.

### 64-bit

```bash 
curl -sSL https://raw.githubusercontent.com/PierreKieffer/htui/master/install/install_htui64.sh | bash
```

## Run 
```bash
htui
```

## Move around
```bash
     -----------------------------
     -        Move around        -
     -----------------------------
     go up               ▲  or 'k'
     go down             ▼  or 'j'
     go to the top       'gg'
     go to the bottom    'G'
     select item         'enter'
     Quit htui           'q'
```
